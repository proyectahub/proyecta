- Archivo HTML autónomo con la animación completa

  - Variante 4: Oferta Especial

  - Variante 6: Educational

- Script para convertir la animación a MP4 profesional
- **3 métodos disponibles:**
  1. FFmpeg (automatizado)
  2. OBS Studio (visual, recomendado)
  3. Windows ScreenRecorder (integrado)

## 🚀 Pasos para Lanzar la Campaña

### PASO 1: Preparar el Video
```
Opción A (Recomendado - OBS Studio):
1. Descarga OBS Studio: https://obsproject.com/
2. Abre mdatos_facebook_video.html en navegador
3. En OBS: Sources → Window Capture → Selecciona navegador
4. Haz clic "Reproducir" en el video
5. En OBS: Click "Start Recording"
6. Espera 45 segundos
7. Stop Recording → El video se guarda en Videos/

Opción B (Windows Integrado):
1. Abre mdatos_facebook_video.html
2. Presiona Windows + G
3. Haz clic en "Grabar"
4. Clic en "Reproducir" en la web
5. Espera 45 segundos
6. Windows + G → Detener grabación
7. El video está en Videos/
```

### PASO 2: Subir Video a Facebook
```
1. Ve a: business.facebook.com
2. Menú → Creador de anuncios
3. Objetivo: Tráfico / Generación de leads
4. Presupuesto: $40-50 USD/día
5. Duración: 7-14 días
6. Sube el video MP4
7. Añade copy de FACEBOOK_ANUNCIOS_VARIANTES.txt
8. Configura público objetivo (ver abajo)
```

## 📊 Estrategia Recomendada

### Semana 1-2:
- Lanza **Variante 1** (Urgencia)
- Presupuesto: $50 USD/día
- Monitorea CTR (meta: >1.5%)

### Semana 3-4:
- Añade **Variante 2** (Residente)
- Pausa si CTR < 1%
- Aumenta presupuesto si CTR > 2%

### Semana 5-6:
- Lanza **Variante 4** (Oferta especial)
- Presupuesto aumentado: $70/día
- Esta suele generar mejor conversión

### Mensual:
- Rota entre variantes
- Reemplaza con bajo desempeño
- Mantén las 2-3 mejores activas

---

## 📈 Métricas a Monitorear

| Métrica | Meta | Acción |
|---------|------|--------|
| CTR (Click-Through Rate) | >1.5% | Si <1%, pausa y ajusta copy |
| CPC (Costo por Clic) | <$0.50 | Si >$0.80, reduce presupuesto |
| Mensajes/Contactos | >15% | Si <10%, revisa CTA |
| Leads/Día | 3-5 | Si <2, aumenta presupuesto |
| CPL (Costo por Lead) | <$10 | Si >$15, cambia audiencia |

## ⚠️ Tips Importantes

1. **Responde en <1 hora**: Si alguien se contacta, responde rápido
2. **Ofrece valor inicial**: "Primera consulta gratis 15 min"
3. **No vendas directo**: Primero entiende su proyecto
4. **Prueba periódicamente**: A/B test diferentes copys
5. **Monitorea diariamente**: Primera semana requiere atención diaria
6. **Escala lo que funciona**: Si una variante genera 10+ leads/día, duplica presupuesto

## 🆘 Troubleshooting

**Problema: Muy pocos mensajes**
→ Aumenta presupuesto $10 más por día
→ Cambia a Variante 4 (Oferta)
→ Revisa que CTA sea clara

**Problema: Muchos mensajes pero sin conversión**
→ Responde más rápido (en < 30 min)
→ Ofrece consulta gratis
→ Pregunta sobre su proyecto antes de precio

**Problema: Video no se ve bien**
→ Descarga nuevo usando OBS Studio
→ Asegúrate resolución sea 400x712 (vertical)
