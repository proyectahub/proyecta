# Figure 3 package

Contents:
- `figure3_target_prioritization.png` and `figure3_target_prioritization.svg`: manuscript-ready figure exports.
- `figure3_structure_inventory.csv`: representative public MeV structures used in the target-evidence panel.
- `figure3_target_prioritization.csv`: rule-based scoring table for study target selection.
- `figure3_source_excerpts.md`: short evidence notes with URLs.
- `figure3_caption.md`: manuscript caption draft.
- `figure3_results.md`: matching Results paragraph draft.
- `raw_sources/`: downloaded JSON/XML source files.
