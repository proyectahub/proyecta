# Figure 1 package

Contents:
- `figure1_epidemiologic_context.png` and `figure1_epidemiologic_context.svg`: manuscript-ready figure exports.
- `figure1_case_milestones.csv`: source-verified outbreak milestones used in panel A.
- `figure1_coverage.csv`: MMR coverage values and threshold used in panel B.
- `figure1_source_excerpts.md`: short evidence excerpts and URLs.
- `figure1_caption.md`: manuscript caption draft.
- `figure1_results.md`: matching Results paragraph draft.
- `raw_sources/`: downloaded HTML/PDF sources.
- `source_text/`: normalized extracted text used by the parser.
