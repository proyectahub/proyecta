📊 CÓMO FUNCIONA

1. Usuario se registra (Email/Password)
2. Crea proyecto (título, meta, descripción)
3. Sistema genera dirección Monero automáticamente
4. Otros usuarios hacen click "Minar para este proyecto"
5. Seleccionan % CPU a donar
6. Navegador genera hashes (200 H/s)
7. Cada 100 hashes, se envían al backend
8. Backend reenvía a SupportXMR pool
9. Pool acumula hashes y minando
10. SupportXMR calcula XMR basado en hashes
11. XMR se envía a dirección del proyecto
12. Cualquiera puede verificar en xmrchain.net

RESULTADO: XMR real en la dirección del proyecto
TIEMPO: Hashes se traducen a XMR en ~10-20 minutos
COMISIÓN: 0% (Non-custodial)
